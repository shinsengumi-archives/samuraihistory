Re: Rekishi Gunzo books - where?

by kitsuno

You can either get them at your local Japanese bookstore (Kinokuniya,
etc.), or you can order them from www.amazon.co.jp
I've ordered books from there before. The books show up in about a
week, and it is 1500 yen for shipping (to the USA). Never had a
problem with them. If you can read Japanese, it is no problem, if
you can't, it might take a few minutes to figure it out, but there is
an english version of the page, it is just a matter of finding the
link. It works just like amazon.com, but you have to create
an 'identity' for the amazon.co.jp - your amazon.com id doesn't
transfer. but it has all the exact same functions as amazon.com -
package tracking, etc. The only possible problem you might come
accross is that you need to be able to type the japanese into the
search engine to find the japanese books you want.
