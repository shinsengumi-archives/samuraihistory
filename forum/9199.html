SamuraiWiki - New Users

by kitsuno

If you are a newly signed up SamuraiWiki user, or signed up a while
ago, but haven't contributed, now would be the time to log in.
Because of the potential for spambots to spam the wiki with false User
IDs, I've begun tagging new users and inactive users who have never
contributed. If you do not remove the template from your user page or
otherwise edit/contribute within 15 days of the notice, your ID will
be blocked. Unfortunately for now this is my only defense against
spambots.

Re: SamuraiWiki - New Users

by kitsuno

Just to clarify - the "SamuraiWiki" is the new Japanese history
"Wiki", which is to say it is the same software and is used the same
as Wikipedia. This also means that you do not have to create an
account to do searches and read the information, etc. Here is the
main page:

http://wiki.samurai-archives.com/

You only need to sign up if you want to edit or add articles. That's
the only reason one would really need or want to sign up. If you
haven't taken a look at the SamuraiWiki yet, it is worth poking
around. Some categories have hundreds of entries, others only 3 or 4.
It's only been online for 2 months, so there is still a lot of work to
go. Some categories that people here might find interesting could be
the "Samurai" Category:

http://wiki.samurai-archives.com/index.php?title=Category:Samurai

The "Bujutsu" Category:

http://wiki.samurai-archives.com/index.php?title=Category:Bujutsu

"Battles":

http://wiki.samurai-archives.com/index.php?title=Category:Battles

And so on. There are about 1100 articles up so far. If you want to
contribute, there are only 2 rules:

1. Cite your sources.
2. If you plagarize from other sources, you will probably be banned
without a warning.
